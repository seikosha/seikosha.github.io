法问项目布置上线调试记录

将自己用springboot+vue+mysql的项目通过docker(前端部分没有)布置上了云服务器，过程中非常折磨，踩得坑很多，终于成功了。只有把它记录下来，以后碰到类似的坑可以避过去。

  • docke后端：

    • 测试启动：

      • mvn spring-boot:run

    • MVN打jar包，布置到与dockerfile同一个文件夹

    • 然后docker build -t fawen_server . 启动

    • 然后run起容器

      • docker run --name fawen_server -d -p 8080:8080 -v /opt/jar/springBootDocker:/log fawen_server

    • 创建以后，进入容器内的mysql

      • source 文件路径(导入sql文件)

    • 安装组件，例如element-ui,axios等

    • 一直保证dev server 在运行的命令：

      • nohup npm run dev >/dev/null 2>&1 & exit

    • 容器内安装vim

      • apt-get update
apt-get install vim

      • 如果想记录日志文件

        • touch my.log
chmod u+w my.log
nohup npm run dev > my.log 2>my.log & exit

    • npm run dev跑起来了，服务器端口也打通了，但是通过浏览器访问不到项目

      • 将/config/index.js中的设置从localhost改为0.0.0.0
